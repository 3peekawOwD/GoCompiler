# GoCompiler

### ENG:
Go compiler written on JS

#### Install

```bash
git clone https://github.com/3peekawOwD/GoCompiler.git
cd GoCompiler/
npm i
```

#### Run Go code

Run `node index.js file.go`

Profit!
